# QRapp – QR Attendance System

Hệ thống điểm danh bằng QR với:

- **Frontend người dùng**: HTML + JavaScript thuần
- **Admin UI**: React (Vite)
- **Backend**: Node.js + Express + Sequelize + MySQL

---

## 1. Yêu cầu môi trường

- Node.js >= 18
- MySQL >= 5.7
- NPM hoặc Yarn

---

```bash
cd backend
cp .env.example .env
